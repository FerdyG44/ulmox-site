ULMOX Säkerhet och moderering

ULMOX arbetar för att erbjuda en trygg och respektfull miljö för alla användare.

Innehållsmoderering

Innehåll som delas på ULMOX kan granskas med automatiska detekteringssystem och modereringsverktyg.

Vi övervakar aktivt och tar bort innehåll som bryter mot våra regler, inklusive:

  • Trakasserier eller mobbning
  • Hets mot folkgrupp eller hatpropaganda
  • Sexuellt eller explicit innehåll
  • Våldsamt eller grafiskt material
  • Spam eller vilseledande innehåll
  • Olaglig aktivitet
  • Utnyttjande av barn eller kränkande beteende

Rapporteringssystem

Användare kan rapportera innehåll direkt i appen.

Rapporterat innehåll granskas och kan bli:

  • Borttaget
  • Dolt från offentliga flöden
  • Eskalerat för ytterligare granskning
  • Använt som underlag för att stänga av eller blockera konton

Blockeringsfunktioner

Användare kan när som helst blockera andra användare.

Blockerade användare kan inte längre interagera med eller visas för den användare som blockerade dem.

AI-assisterad detektering

ULMOX kan använda automatiserade system för att upptäcka potentiellt osäkert eller olämpligt innehåll innan det sprids vidare.

Innehåll som flaggas av automatiserade system kan genomgå ytterligare granskning.

Användarsäkerhet

Vi uppmuntrar respektfull kommunikation och positiva interaktioner.

Konton som upprepade gånger bryter mot våra regler kan stängas av tillfälligt eller permanent.

Barnsäkerhet

ULMOX har nolltolerans mot material som visar sexuella övergrepp mot barn (CSAM) och mot exploaterande beteende.

Allt upptäckt innehåll eller aktivitet kopplat till CSAM tas bort omedelbart och kan rapporteras till relevanta myndigheter.
